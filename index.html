작업 기준 · 요청 / 마감 스케줄
Guideline
작업 기준
요청 가능 범위
3D 또는 그래픽이 얹어지는 건에 한해 작업 요청을 받습니다.
웹디자인팀 내재화 범위
단순 모션 / 컷 편집은 웹디자인팀에서 진행합니다. · 그래픽 없이 단순 모션 작업 · 컷 편집 (컷 전환 + 텍스트 등장 위주의 GIF 등)
시안 요청 지양
단순 모션의 여러 시안 요청(속도·위치 세밀 조정 등)은 지양합니다. 퀄리티 향상과 비용 절감에 에너지를 집중할 수 있도록, 단순 영상 시안을 여러 개 뽑고 피드백을 주고받는 방식은 지양합니다.
